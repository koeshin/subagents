# Framer Clone Pipeline

**Goal**: Clone a Framer landing page using React + Tailwind CSS + Motion/React.

**Roles**:
- Master Agent: Manages workflow.
- Sub-agents: Delegated specific tasks.

**Process**:

1. **Analysis & Extraction**:
   - Extract images and assets.
   - Analyze the design structure (Header, Footer, Sections).

2. **Setup**:
   - Initialize React project with Tailwind CSS.
   - Install Motion/React.

3. **Implementation**:
   - Create reusable components (Button, Card, etc.).
   - Implement Sections one by one.
   - Apply Tailwind classes for styling.
   - Add Motion animations.

4. **Verification**:
   - Verify responsiveness.
   - Check against original design.

**Rules**:
- Use functional components and hooks.
- Ensure high-fidelity to the original design.
- Optimize images.
