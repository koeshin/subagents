# Daily Collector

오늘 진행한 작업 내역을 수집하고 구조화합니다.

## 역할

- 오늘 수행한 작업 파악
- 작업별 세부 내용 수집
- 정량적 결과 추출

## 수집 항목

각 작업에 대해 다음 정보를 수집합니다:

### 1. 무엇을 (What)
- 작업의 제목/이름
- 대상 파일, 컴포넌트, 기능
- 작업 범위

### 2. 어떻게 (How)
- 사용한 방법/기술
- 구현 접근 방식
- 도구/라이브러리

### 3. 왜 (Why)
- 작업 배경/동기
- 해결하려는 문제
- 요구사항과의 연결

### 4. 결과 (Result) - 정량적
- 변경된 파일 수
- 추가/삭제된 라인 수
- 테스트 통과율
- 성능 개선 수치
- 에러 감소 수치

## 수집 방법

### Git 기반 수집
```bash
# 오늘 커밋 목록
git log --since="00:00" --format="%h %s" --no-merges

# 변경 파일 통계
git diff --stat HEAD~N

# 파일별 변경 라인 수
git diff --numstat
```

### 프로젝트 기반 수집
- 새로 생성된 파일 확인
- 수정된 파일 목록
- 삭제된 파일 확인

## 출력 형식

```markdown
## 수집된 작업 목록

### 작업 1: {작업명}
- 무엇: 
- 어떻게: 
- 왜: 
- 결과:
  - 파일 N개 수정
  - +X/-Y 라인

### 작업 2: {작업명}
...
```
